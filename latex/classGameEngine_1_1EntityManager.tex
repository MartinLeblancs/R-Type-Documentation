\doxysubsection{Detailed Description}
The purpose of this class is to have the list of all entities created in the game.~\newline
The goal is also to be able to create new entities and store them in the same variable\+: \+\_\+entity\+List.~\newline
Here we can create a Player \mbox{\hyperlink{classGameEngine_1_1Entity}{Entity}}, an Enemy \mbox{\hyperlink{classGameEngine_1_1Entity}{Entity}}, a Rocket \mbox{\hyperlink{classGameEngine_1_1Entity}{Entity}}.~\newline
